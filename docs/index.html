What's inside

Everything you need to read, tune & reflash an ECU

Two apps, one theme and sidebar: ECU Studio for reflashing and map editing, SocketSpy for CAN analysis. Everything runs on your machine.

🔌

MPPS Flashing

Read & write ECU ROM over MPPS V21 (FTDI/CH340/CP2102). libusb on Linux, no proprietary driver needed. Background-threaded with live progress.

🗃️

Virtual Hex Editor

Smooth hex view for 2–6 MB ROMs with map-region overlays (planned: 60fps QPainter virtual scrolling).

📊

Bosch Map Editor

Read/write Kf_Xs16_Ys16_Ws16 maps, apply percentage changes to Stage 1 maps with positive-only scaling.

🔎

Heuristic Map Finder

Detect 2D maps in a ROM without an A2L, scored by axis monotonicity and data smoothness.

📄

A2L / DAMOS Parser

Parse ASAP2 CHARACTERISTIC/AXIS_PTS/RECORD_LAYOUT, resolve cross-references (Bosch & PSA dialects).

✅

Checksum Correction

Auto-correct ECU checksums before writing, with read-back verification.

≠

ROM Compare

Byte-level diff between two ROMs, grouped into changed map intervals.

Ⓖ

Git Versioning

libgit2-backed ROM history — commit any state, browse the log, and restore any previous version in one click. Instant rollback, no cloud.

🚗

ECU Catalog & Vehicle Templates

Built-in catalog (EDC16C34, EDC17, ME7, MED17…) and one-click Stage 1 / Pop&Bang / depollution presets per vehicle.

📡

SocketSpy CAN Analysis

Bundled SocketSpy app: live CAN monitor, DBC decoding, UDS, Lua scripting — watch the bus during a flash.

🪟

Detachable Panels

Same icon-sidebar UX as SocketSpy: drag or right-click any panel to float it as its own window.

🔒

100% Local

No account, no telemetry, no cloud. Everything runs on your machine.
