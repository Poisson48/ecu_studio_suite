Live UI — Qt6 dark theme

ECU Studio — demo gallery

Real screenshots of every panel, captured straight from the Qt6 build in simulation mode. No hardware required — everything runs 100% locally.

ECU Studio — Hex

Hex editor

Fast byte-level viewer with address jump, search and live map-range highlighting. Edits flow straight into the shared ROM document and notify every other panel.

ECU Studio — Maps

Map editor

Visualize and edit scalar, curve and 2-D/3-D Bosch maps with RPM/load axes and a heat-mapped grid. Apply a percentage delta or a full Stage 1 recipe in one click.

ECU Studio — Checksum

Checksum correction

Compute and patch checksums (Sum32, CRC-32, …) per region. The results table flags mismatches in red and the journal logs every verification and fix.

ECU Studio — Compare

ROM compare

Side-by-side byte diff of two ROMs. The modified-ranges table lets you jump any delta straight to the hex editor for inspection.

ECU Studio — AutoMods

AutoMods

Apply named calibration patches from JSON recipes — Stage 1, EGR off, DTC removal, speed-limiter changes — with batch apply and one-click revert.

ECU Studio — Git

Git versioning new

libgit2-backed ROM history: commit any state, browse the full commit log, and restore (roll back) any previous version straight into the editor. Instant rollback, no cloud.

ECU Studio — A2L

A2L browser

Parse ASAP2 .a2l files and browse characteristics and measurements by name, type, address and unit — with instant filtering and double-click to hex.

ECU Studio — MPPS

MPPS flashing

Detect the MPPS V21 programmer, pick K-Line or CAN, then read or write the full ROM over USB with a live progress bar. Simulation mode needs no hardware.

ECU Studio — Project

Project manager

Organise work into projects with ROM path, ECU type, vehicle metadata and notes. Opening a project wires its directory into the Git versioning panel automatically.
